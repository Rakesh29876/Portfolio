Introduction:

This project presents a comprehensive analysis of a pizza chain's sales performance using SQL (MySQL). The primary objective is to uncover trends in customer behavior, product preferences, and sales patterns to support data-driven decision-making.

Data Preparation:

  1. Data Source:

    Raw transactional dataset of a fictional pizza chain.

  2. Data Platform:

    MySQL for query execution.

  3. Data Cleaning:

    The initial setup for the database pizza is shown in the queries. While not explicitly detailed as "cleaning steps" in the slides, typical data preparation would involve null handling, column renaming, and formatting dates as implied by the queries.

SQL Queries and Analysis:

The analysis involved running various SQL queries to extract key insights:

  1. Total Number of Orders Placed:

    Query: SELECT count(*) from orders;

  2. Total Revenue Generated from Pizza Sales:

    Query: SELECT round(sum(order_details.quantity * pizzas.price),2) FROM order_details JOIN pizzas ON order_details.pizza_id = pizzas.pizza_id;

  3. Highest-Priced Pizza:

    Query: Select pizza_types.name, pizzas.price from pizza_types join pizzas on pizza_types.pizza_type_id=pizzas.pizza_type_id order by pizzas.price desc limit 1;

  4. Distribution of Orders by Hour of the Day:

    Query: select hour(time)as hours, count(order_id)as order_count from orders group by hours;

  5. Category-Wise Distribution of Pizzas:

    Query: Select category as pizza_name,count(name)as total_count from pizza_types group by category;

KPIs:

  1. Total Orders: 21,350.
  2. Total Revenue: 817,860.05.
  3. Highest Priced Pizza: The Greek Pizza ($35.95).
  4. Most Common Pizza Size: Large (L) with 18,526 orders.

Key Insights:

Here are the key insights and tools used, based on the provided presentation:

  1. Popular Pizza Types: "The Classic Deluxe Pizza" (2453 quantity), "The Barbecue Chicken Pizza" (2432 quantity), and "The Hawaiian Pizza" (2422 quantity) are consistently top sellers by quantity.
  2. High-Revenue Generators: While popular by quantity, specific pizzas like "The Thai Chicken Pizza" ($43434.25 revenue), "The Barbecue Chicken Pizza" ($42768 revenue).
  3. Peak Order Hours: There is a clear distribution of orders by hour, with peaks around 12 PM (2520 orders) and 1 PM (2455 orders), which is crucial for efficient staffing and operations.
  4. Consistent Daily Averages: The average number of pizzas ordered per day remains fairly consistent (e.g., 162.00 on Jan 1, 2015), providing predictability for inventory management.
  5. Revenue Growth Tracking: The analysis of cumulative revenue over time provides a clear visual of the business's financial growth trajectory.
