Introduction:

This project presents an analysis of a Spotify dataset using SQL. The primary objective is to extract insights related to songs and artists from the dataset.

  1. Data Source:

    Raw transactional dataset of a spotify.

  2. Data Platform:

  3. Database Setup:

    The initial setup involved creating a database named 'spotify' and then using it for subsequent operations. The data is queried from a table named.

SQL Queries and Analysis (based on the Spotify project images):

The analysis involved running various SQL queries to extract key insights from the spotify tables:

  1. Count of Songs by Each Artist:

    select artists,count(name) as song_count from spotify.top2018 group by artists order by song_count;

  2. Top 5 Most Energetic Songs:

    select name, artists, energy from spotify.top2018 order by energy limit 5;

  3. Longest Song for Each Artist:

    with rankedsongs as(select name, artists, duration_ms, row_number() over(order by duration_ms desc)as ranks from spotify.top2018) select name, artists, duration_ms from rankedsongs where ranks=1;

  4. Calculate Running Total of Duration_ms for Each Artist:

    select id,name, artists, duration_ms, sum(duration_ms) over(partition by artists order by id) as running_total_duration from spotify.top2018;

  5. Tempo Distribution within Time Signatures:

    SELECT time_signature, AVG(tempo) AS average_tempo, STDEV(tempo) AS standard_deviation_tempo FROM spotify.top2018 GROUP BY time_signature ORDER BY time_signature;

Key Performance Indicators (KPIs) for the Spotify Project:

  1. Average Song Tempo: The average tempo across all songs in the top2018 dataset is approximately 119.90.
  2. Longest Song Duration: The longest song identified is 'Te Bot? - Remix' by Nio Garcia, with a duration of 417920 milliseconds.
  3. Average Tempo by Time Signature: Time Signature 3: Average tempo of 108.029.
  4. Artists with Most Diverse Tempo Range: Drake: Tempo range of 77.814.

Key Insights:

  1. Artist Song Count Variability:

    While many artists appear to have only one song in the top2018 dataset (e.g., Juice WRLD, G-Eazy, Sofia Reyes), a more detailed analysis of the full result set for "Count of Songs by Each Artist" would reveal which artists have multiple entries, indicating their prominence in the dataset.

  2. Dominant Songs and Artists:

    Specific songs and artists are clearly identifiable, such as 'God's Plan' by Drake and 'SAD!' by XXXTENTACION, highlighting popular tracks from the top2018 list.

  3. Average Tempo as a Characteristic:

    The overall average tempo of songs in the top2018 dataset hovers around 119.90 BPM, providing a general understanding of the musical pace.

  4. Tempo Distribution Across Time Signatures:

    There are distinct average tempo values for different time signatures (e.g., 108.029 for time signature 3, 120.111 for time signature 4), suggesting characteristic tempos associated with different rhythmic structures.
