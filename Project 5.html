Introduction:

This project presents a comprehensive analysis of World Cup data using SQL (MySQL). The primary objective is to uncover trends in match performance, team success, and player statistics to provide data-driven insights into World Cup history.

Data Preparation:

  1. Data Source:

    Raw transactional dataset related to the World Cup.

  2. Data Platform:

    MySQL for query execution.

  3. Database Setup:

    The initial setup for the database involved creating a database named worldcup and then using it for subsequent operations.

SQL Queries and Analysis:

The analysis involved running various SQL queries to extract key insights:

  1. Total Number of Matches Played in the World Cup:

    Select count(*) as total_matches from world_cup_matches; [cite: 1].

  2. List All World Cup Winners:

    select year, world_cups.winner from world_cups order by year; [cite: 1].

  3. Count of Matches Per Year:

    Select year,count(world_cup_matches.date)as count_matches from world_cup_matches group by year order by count_matches; [cite: 1].

  4. Number of Goals Scored Per World Cup:

    select year,'goals scored' from world_cups; [cite: 1].

  5. Average Goals Per Match (World Cup History):

    SELECT year, ROUND(AVG('home_goals' + 'away goals'), 2) AS avg_goals_per_match FROM world_cup_matches group by year; [cite: 1].

KPIs:

  1. Total Matches Played: 900.
  2. Most Successful Teams (by Qualified Teams): Italy, Brazil, France, Spain, and Germany each had 32 qualified teams.
  3. Most Matches Hosted by a Country: Italy, France, Brazil, Mexico, and Germany each hosted 2 matches.
  4. Highest Goal-Scoring Match: Switzerland vs. Austria with 12 total goals on 1954-06-26.

Key Insights:

  1. Total Matches & Historical Scope:

    The World Cup has a rich history with a significant volume of matches played, totaling 900 events.

  2. Individual Excellence (2022):

    The 2022 World Cup data highlights top individual goal-scoring prowess, with Cristiano Ronaldo, Lionel Messi, Robert Lewandowski, Neymar, and Romelu Lukaku standing out as leading scorers.

  3. FIFA Ranking Accuracy:

    The 2022 FIFA rankings show Brazil, Belgium, and Argentina as the top three teams, reflecting their expected high performance.

  4. High-Scoring Encounters:

    Historical data reveals exceptionally high-scoring matches, such as Switzerland vs. Austria in 1954 with 12 goals, indicating periods of highly offensive play.

  5. Host Nation Contribution:

    Multiple countries have played significant roles in hosting the World Cup, with Italy, France, Brazil, Mexico, and Germany each hosting 2 matches.
