Introduction

This project provides a detailed analysis of Diwali sales using Python in Jupyter Notebook. It explores consumer behavior, product preferences, and regional sales patterns using data visualization and pandas-based aggregation techniques.

Objective

  • Analyze customer purchase behavior during the Diwali festival season using transactional data.
  • Perform data cleaning to remove null values and irrelevant columns for more accurate analysis.
  • Explore sales trends across different age groups, genders, and marital statuses.
  • Identify the top-performing product categories and subcategories based on sales volume.
  • Uncover city-tier-wise customer preferences to support location-based marketing strategies.

Methodology

  1. Data Import

    The Diwali Sales dataset was imported into the Jupyter Notebook environment using the pandas library. The dataset was in CSV format and was read using the pd.read_csv() function. Initial exploration included checking the shape, data types, column names, and sample records using functions like .head(), .info(), and .describe().

  2. Data Cleaning.

    Data cleaning is a crucial step in preparing the Diwali sales dataset for accurate analysis. It involves identifying and resolving issues such as missing values, irrelevant columns, and inconsistent data types that can distort insights and lead to incorrect conclusions.

  3. Eda Exploratory Data Analysis.

    Total Amount vs gender:

    Analyzing the relationship between gender and total purchase amount helps in understanding consumer behavior and targeting marketing strategies effectively.

    Age Group:

    Understanding which age groups contribute the most to total Diwali sales provides valuable insights into target demographics and helps tailor marketing strategies accordingly.

    Total Amount/Sales:

    This analysis identifies the top 10 customers with the highest total purchase amounts during the Diwali sales period.
