Introduction:

This project presents a comprehensive analysis of a restaurant's performance using SQL (MySQL). The primary objective is to uncover trends in menu item popularity, pricing, order patterns, and customer spending habits to support data-driven decision-making.

Data Preparation:

  1. Data Source:

    Raw transactional dataset of a restaurant.

  2. Data Platform:

    MySQL for query execution.

  3. Database Setup:

    The initial setup for the database involved creating a database named restaurant and then using it for subsequent operations.

SQL Queries and Analysis:

The analysis involved running various SQL queries to extract key insights from the menu_items and order_details tables:

  1. Find the number of items on the menu:

    Select count(*) from menu_items; [cite: 1].

  2. How many Italian dishes are on the menu?:

    Select count(*) from menu_items where category='italian'; [cite: 1].

  3. What are the least and most expensive Italian dishes on the menu?:

    SELECT MIN(price) AS minimum_price, MAX(price) AS maximum_price FROM menu_items WHERE category = 'italian'; [cite: 1].

  4. What is the average dish price within each category?:

    Select category, round(avg(price),1) from menu_items group by category; [cite: 1].

  5. How many orders were made within this date range?:

    SELECT COUNT(distinct order_id) FROM order_details; [cite: 1].

KPIs:

  1. Total Menu Items: 32.
  2. Most Expensive Menu Item: $19.95.
  3. Total Unique Orders: 5370.
  4. Average Price per Category: Italian ($16.8), Asian ($13.5), Mexican ($11.8), American ($10.1).

Key Insights:

  1. Popularity by Category:

    Italian and Mexican categories have the highest number of dishes (9 each), suggesting a focus on these cuisines.

  2. Top-Selling Items:

    Hamburger (American), Edamame (Asian), and Korean Beef Bowl (Asian) are among the most frequently purchased items, regardless of category.

  3. Consistent Menu Pricing Strategy:

    The restaurant maintains a relatively consistent pricing ceiling, with the most expensive item across all categories priced at $19.95.

  4. Customer Ordering Patterns:

    The restaurant processed 5370 unique orders within a little over two months (Jan 1, 2023 - Mar 9, 2023), indicating consistent activity.
