Introduction

This project focuses on building an interactive Power BI dashboard to monitor Adverse Events (AEs) reported on social media platforms such as Twitter and Reddit. The goal is to analyze user posts, identify serious and unexpected adverse events, track sources, and provide insights for better drug safety monitoring and pharmacovigilance.

Objective

  • To track total posts and sources reporting adverse events.
  • To detect serious and unexpected AEs from social media mentions.
  • To perform geo-location and source analysis for identifying high-risk regions.
  • To evaluate model accuracy and false predictions in AE detection.
  • To provide an interactive dashboard for stakeholders with filtering and drill-through capabilities.

Methodology

  1. Data Collection.

    Gathered raw data from Twitter and Reddit sources. Extracted multiple tables such as M Processed MedDRA, M Processed Serious, M Processed XML, MI_PQC, Source Master, Search Configuration.

  2. Data Cleaning.

    Removed duplicate posts and irrelevant records. Standardized inconsistent fields (e.g., location names, source names). Handled missing values in demographic and event-related fields.

  3. Data Transformation.

    Created calculated columns for Severity, Unexpected Flags, Drug Names, Symptom Categories, and Event Classification. Converted date/time fields into proper formats for time-series analysis. Normalized categorical fields (e.g., Twitter vs Reddit posts).

    Data Modeling:

    Established relationships between key tables (Posts ↔ Sources ↔ MedDRA ↔ Serious Events). Built a Star Schema model with fact tables (AE reports, PQC cases) and dimension tables (Source, Location, Configurations).

    Insight Extraction:

    Analyzed which drugs/symptoms are most frequently reported. Identified locations with highest serious AEs. Compared Twitter vs Reddit contributions.

Result

  1. The dataset contained social media posts from Twitter and Reddit, enriched with metadata such as location, source type, drug/symptom details, and event severity.
  2. A total of serious adverse events (SAEs) were successfully identified and categorized using MedDRA mappings.
  3. The analysis revealed Twitter as the dominant source, contributing the majority of posts compared to Reddit.

Summary

  • Tools Power BI (Power Query, DAX, Interactive Dashboards)
  • Objective To monitor social media platforms (Twitter & Reddit) for detecting and analyzing adverse drug events, enabling better pharmacovigilance and decision-making.
